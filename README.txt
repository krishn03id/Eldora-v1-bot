
Telegram RPG Bot - EldoriaBot
=============================
Features:
- XP, Gold, Leveling
- Daily rewards and duels
- DiceBear avatars
- SQLite storage
- Leaderboard

Setup:
1. Replace YOUR_BOT_TOKEN_HERE in main.py with your BotFather token.
2. Run `pip install -r requirements.txt`
3. Start the bot with `python main.py`
